本地上传到GitHub
	1.在客户端点击create,新建一个仓库
	2.把项目复制到仓库（也可以直接把项目所在文件夹设置成仓库）
	3.点击发布按钮，发布到github
	4.如果是在原有的文件上修改，修改完之后就会出现change<变化>,直接加备注提交就行
GitHub克隆到本地
	1.在GitHub上点击克隆按钮
	2.如果是新项目就会出现选择文件路径的窗口
	3.如果本地已经存在的文件就不会有新窗口，如果有修改就直接点击发布按钮，此时的发布按钮箭头向下标识
	  克隆到本地
删除仓库
	1.在GitHub上选中要删除的仓库
	2.进入仓库点击seting
	3.把页面拉倒最下端，点击delete，填写仓库名，确认删除
